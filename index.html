Creating columns using Css

Column using display inline-block

collumn 1
collumn 2

Html: 

		
			<div class="collumns_container">
				<div class="col col2">collumn 1</div>
				<div class="col col2">collumn 2</div>
			</div>

Style: 

		

			.collumns_container {
			    display: block;
			    max-width: 1200px;
			    white-space: nowrap;
			    font-size: 0;
			    padding: 15px;
			    margin: 0 auto;
				background: gray;
			}

			.collumns_container .col {
				display: inline-block;
				vertical-align: middle;
			 	box-sizing: border-box;
			 	white-space: normal;
				border: 0;
				margin: 0;
				font-size: 14px;
			}

			.collumns_container .col.col2 {
				width: 50%;
			}

Column using display flex

collumn 1
collumn 2

Html: 

		
			<div class="collumns_container_flex">
				<div class="col col2">collumn 1</div>
				<div class="col col2">collumn 2</div>
			</div>

Style: 

		
			.collumns_container_flex {
				    max-width: 1200px;
				    margin: 0 auto;
				    display: flex;
				    flex-flow: row wrap;
				    padding: 15px;
					background: #d8d8d8;
				}

				.collumns_container_flex .col {
					flex: 0 0 100%;

				}

				.collumns_container_flex .col.col2 {
					flex: 0 0 50%;
				}
